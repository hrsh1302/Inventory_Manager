Welcome to the Inventory Manager Program

This program is an Inventory Manager for a business willing to digitalized its inventory. It is capable of running the CRUD operations (Create / Read / Update / Delete).
This program uses Spring as the Java framework, MySQL as the DBMS to store the data, Bootstrap for the HTML/CSS framework used, Thymeleaf as the template engine.
Check the navbar on your left to check and try its functionalities. Have fun :)
